## Screenshot Guidelines

1. **Resolution:** Minimum 1920x1080 pixels
2. **Format:** PNG preferred (no compression artifacts)
3. **Content:** Use realistic demo data (no "test123" or "Lorem ipsum")
4. **Theme:** Capture in both light and dark mode if possible
5. **Browser:** Hide browser chrome if possible, or use clean screenshots

## How to Capture Screenshots

### Using Browser DevTools
1. Open Chrome DevTools (F12)
2. Click the device toolbar icon (Ctrl+Shift+M)
3. Set resolution to 1920x1080
4. Use "Capture screenshot" from the three-dot menu

### Using Third-Party Tools
- [CleanShot X](https://cleanshot.com/) (macOS)
- [ShareX](https://getsharex.com/) (Windows)
- [Flameshot](https://flameshot.org/) (Linux)
